Web Design

Web design refers to the design of websites. It usually refers to the user experience aspects of website development rather than software development.

Web Development

Web development is closely related to the job of designing the features and functionality of websites and apps, but the “web development” term is usually reserved for construction of websites and apps.

Responsive Design

Responsive web design (RWD) is a web design approach to make web pages render well on all screen sizes and resolutions while ensuring good usability. It is the way to design for a multi-device web.
